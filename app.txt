Se detallan a continuación algunos comandos básicos para el uso de git por consola
1. git add .   // Agrega al stage todos los archivos que se hayan creado en el proyecto
2. git commit -m "Mensaje del commit"  // Confirma definitivamente los archivos agregados al stage
3. git push origin main // Envia los archivos del respositorio local al respoitorio remoto
